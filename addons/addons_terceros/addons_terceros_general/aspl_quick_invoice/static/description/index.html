Odoo 14 Quick Invoice (Fast Invoice) (Community)

This module allow user to create new invoice order from Quick Invoice.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • Configure journals for create invoice order from Quick Invoice.

  • When you click on invoicing menu you can see one button "Quick Invoice" in 'Customer Invoices' order list view.

  • When you click on 'Quick Invoice' button open new view for create invoice for customer.
  • you can search products and scan products via barcode and add into cart.
  • Also you can set customer.
  • As per below image you can also select cart line and you can set quantity, click on discount button and set disount for selected line and click on price button and set product price for selected line also.
  • Payment line will be visible as per below image and you can split payment in different journals and base on due payment amount create invoice order as paid,partial and draft.
  • In footer part, you can see whole invoice order information like total amount, untax amount, taxes etc.



  • When invoice order successfully created then you can see one message in popup and with invoice order number.

  • You can also see one icon after 'Invoice Date' field, when you click on that icon you can see invoice orders which you create from quick invoice till you reload page.

  • Also when you click on invoice order number, it will redirected to that invoice order record.
